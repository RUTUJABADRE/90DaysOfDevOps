## Process Management

1. ps — View running processes
2. top — Real-time process monitoring
3. htop — This command use to shoe the processes in Interactive process viewer and more friendly than top command
4. kill — Stop a process
5. systemctl — Manage system services operation like start, stop, status, restart
6. pgrep - This command used to search process by name and return their Process ID (PID)
7. lsof - shows which process is using a file, port, or network connection

---

## File system

1. ls — List files and directories
2. cd — Change directory
3. pwd — Show current directory
4. mkdir — Create directory
5. rm — Delete file or directory
6. cp — Copy files
7. mv — Move or rename files
8. touch — Create empty file
9. cat — View file content
10. head — View first lines
11. tail — View last lines
12. du — Check folder size
13. df — Check disk space
14. chmod — Change file permission
15. chown — Change file owner
16. find — Find files
